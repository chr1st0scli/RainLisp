# +
```scheme
(+ num1 num2 . nums)
```
Returns the sum of numeric values. It accepts two or more values.

```scheme
(+ str1 str2 . strs)
```
Returns the concatenation of string values. It accepts two or more values.

## Examples
```scheme
(+ 1 2 3 4)
```
-> *10*

```scheme
(+ "Rain" "Lisp")
```
-> *"RainLisp"*
